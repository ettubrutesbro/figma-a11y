# figma-a11y

while running this plugin, selecting 2 figma nodes (shapes, text, etc.) that have fill colors will tell you the WCAG contrast ratio, and whether that falls into WCAG's AAA or AA standard. 

references: 
* https://medium.com/san-francisco-digital-service/color-accessibility-beyond-a-pass-fail-2ea19be4b3c1 (ironically, this article [eloquently!] states a case for not always adhering to WCAG standards - they don't always reflect typographic differences)  
* https://www.w3.org/WAI/WCAG21/quickref/?versions=2.0&source=post_page---------------------------#qr-visual-audio-contrast-contrast
